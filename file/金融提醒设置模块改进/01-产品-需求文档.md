## 1. 业务背景

Bgx 是一个金融数据分析与提醒系统，主要功能包括：
- 股票和基金数据的采集、存储与展示
- 基于价格和涨跌幅的价格提醒功能
- 风险提醒自动检测（工作日 11:30 和 14:30 触发）
- 基金分析功能

**提醒设置模块**允许用户创建、管理个性化的价格监控提醒，支持批量操作，并与仪表盘和风险提醒页面联动。

## 3. 用户操作流程

### 3.1 创建提醒流程

```
1. 进入"提醒列表"页面
2. 点击"创建提醒"按钮
3. 选择模式：
   ├─ 快速模式 → 选择标的类型 → 选择标的 → 点击"一键订阅"
   └─ 自定义模式 → 选择标的类型 → 选择标的 → 选择提醒类型 → 设置目标值 → [展开高级选项] → 点击"创建提醒"
4. 查看创建结果弹窗
5. 成功则跳转列表页
```

### 3.2 批量管理流程

```
1. 进入"提醒列表"页面
2. 勾选要操作的提醒（支持全选）
3. 选择操作：批量删除 / 批量启用 / 批量禁用
4. 确认操作
5. 查看操作结果
```

### 3.3 仪表盘联动流程

```
1. 在仪表盘查看活跃提醒数量（可点击）
2. 在仪表盘查看未读消息栏（风险提醒数+价格提醒数）
3. 点击任意卡片跳转到对应列表
```

## 4. 联动关系

### 4.1 与仪表盘的联动

| 仪表盘区域 | 行为 |
|-----------|------|
| 活跃提醒卡片 | 显示数量，点击跳转提醒列表 |
| 已触发提醒 | 显示数量 |
| 未读消息栏 | 显示风险提醒+价格提醒未读数，点击跳转 |

### 4.2 与风险提醒的联动

| 场景 | 行为 |
|------|------|
| 订阅风险提醒 | 自动创建对应的价格提醒（涨跌幅1%） |
| 价格提醒触发 | 生成风险提醒记录 |

## 6. 优先级汇总

### P0 - 必须完成

- 模式切换 UI（顶部大按钮）
- 快速模式表单（极简版，1%固定）
- 自定义模式表单（分区清晰）
- 批量删除、批量启用/禁用
- 仪表盘联动（活跃提醒可点击、未读消息栏）

### P1 - 重要

- 批量创建增强
- 风险提醒订阅联动
- 提醒列表增强（筛选、排序）

### P2 - 可延后

- 高级筛选条件
- 提醒分组功能
- 导出功能
- 提醒名称、备注等可选字段
