# 数值精度修改工作文档

## 一、需求说明

### 精度标准
| 类别 | 字段示例 | 精度 | 数据库类型 |
|------|----------|------|-----------|
| **净值类** | 单位净值、昨日净值、累计净值、涨跌额 | 4位小数 | DECIMAL(10,4) |
| **百分比类** | 涨跌幅、日/月/周/年增长率 | 2位小数 | DECIMAL(10,2) |
| **价格类** | 股票开盘价、最高价、最低价、收盘价 | 2位小数 | DECIMAL(10,2) |
| **金额类** | 成交额、市值、基金规模 | 2位小数 | DECIMAL(20,2) |

## 四、BigDecimal使用规范

### 4.1 创建方式
```java
// 推荐：使用String构造器避免精度问题
BigDecimal nav = new BigDecimal("1.2345");

// 避免：使用double构造器会有精度问题
BigDecimal bad = new BigDecimal(1.2345); // 实际是1.23449999...
```

### 4.2 运算后设置精度
```java
// 运算后设置精度
BigDecimal result = nav.multiply(quantity).setScale(4, RoundingMode.HALF_UP);
```

### 4.3 数据库映射
```java
// MyBatis-Plus
@Column(precision = 10, scale = 4)
private BigDecimal nav;
```
