## 数值精度标准（供参考）

| 类别 | 示例 | 精度 | 数据库类型 |
|------|------|------|-----------|
| 净值类 | 单位净值、昨日净值、累计净值 | 4位小数 | `DECIMAL(10,4)` |
| 百分比类 | 涨跌幅、日/月/年增长率 | 2位小数 | `DECIMAL(10,2)` |
| 价格类 | 开盘价、最高价、最低价、收盘价 | 2位小数 | `DECIMAL(10,2)` |
| 金额类 | 成交额、市值、基金规模 | 2位小数 | `DECIMAL(20,2)` |
| 数量类 | 持股数量、用户数 | 整数 | `BIGINT` |

## BigDecimal 注意事项

```java
// ✅ 使用 String 构造器避免精度问题
BigDecimal nav = new BigDecimal("1.2345");

// ❌ 避免使用 double 构造器
BigDecimal bad = new BigDecimal(1.2345); // 精度丢失！

// ✅ 运算后设置精度
BigDecimal result = nav.multiply(quantity).setScale(4, RoundingMode.HALF_UP);

// ✅ 比较使用 compareTo
if (value.compareTo(BigDecimal.ZERO) > 0) { }
```
